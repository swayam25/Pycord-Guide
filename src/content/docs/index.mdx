## Quick Start

Get your first bot running in minutes:

```py
import discord

bot = discord.Bot()

@bot.slash_command(description="Say hello!")
async def hello(ctx):
    await ctx.respond("Hello from Pycord! ðŸ‘‹")

bot.run("YOUR_TOKEN_HERE")
```

Just **three simple steps**:

1. Install Pycord: `pip install py-cord`
2. Create your bot on the [Discord Developer Portal](https://discord.com/developers/applications)
3. Copy the code above, add your token, and run!
